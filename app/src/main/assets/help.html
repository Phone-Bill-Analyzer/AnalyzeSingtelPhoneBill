Getting Started

Thank you for downloading this app.

Now, you have the option to really understand your phone bill and analyze the usage also. Just upload your mobile phone bill and start your analysis.

Upload Mobile Bill:

On the main screen, select option: New Bill and upload new bill.
Currently we are supporting only SingTel Post Paid mobile bills in PDF formats.
The PDF file should NOT be encrypted.
You must download the bill in PDF format from SingTel. Scanned copy of a printed bill will NOT work.

Note :
  • If you are unable to browse files, please install a good file browser app.
  • In the file browser screen, activate 'Display advanced devices' option to see all files and folders

Analysis

After you have successfully uploaded your bill, you can see your bill details on the main screen. Select your bill to start analyzing. As of now, you can see the following analysis:
  • Itemised bill
  • Top 5 callers by Name
  • Call costs by Contact groups
In future, we are planning to show more analytics.

Contact Groups

You can create contact groups for e.g. Family, Friends, Co-Workers etc.
Important Points to note:
  • You can assign a contact to multiple groups.
  • You can update contact names, groups and group members after uploading the bill also. Hence, the bill analysis will continue to show old names and old groups.
  • If you have modified your contacts and groups, then please reload the contact details from main screen.

Limitations

  • As of now, we support only SingTel Post Paid bills. We will support more in future
